# Travelling Salesman Problem

A Genetic Algorithm built in [Rust](https://www.rust-lang.org/) to find an optimum to the Travelling Salesman Problem.

## Installation
```
curl https://sh.rustup.rs -sSf | sh
```

## Usage
```
cargo run
```

## Selection Schemes
 - Tournament
 - Proportional Roulette
 - Random
 - Elitism
